# 06b — Future Egg Backlog

Bench-depth alternatives for the six MVP eggs (in case any don't survive
playtesting), additional Hub 1 / 2 ideas if egg count expands, and
starter eggs for Hubs 3–5 so those hubs aren't blocked when added.

All ideas Tier 1 deliverable: gaze trigger, audio narration, illustrated
overlay, no reading required. All defended against an Indiana Kindergarten
standard.

## Eggs I'd resist

| Idea                                | Why                                                                               |
|-------------------------------------|-----------------------------------------------------------------------------------|
| Caucus race as physics              | Fun in the book; no clean K hook that isn't a stretch.                            |
| "Drink Me" mixing (fractions, pH, ratios) | Every angle is above K. Use the bottle as measurement-vocab only and stop reaching. |
| "Curiouser and curiouser" as grammar | Cute for adults; not a K standard. Save for higher tiers.                         |
| Naming "gravity"                    | K.PS.1 is about *investigating* motion, not naming forces. Park for Tier 2.        |

## Authoring rule

When promoting an egg from this backlog into a hub:
- Re-confirm the standard code against the live IDOE PDF.
- Run it past the same Indiana K teacher who approved the MVP six.
- Update [03-modular-architecture.md](03-modular-architecture.md) only if
  the egg requires a new interaction tier or schema change. Otherwise
  the addition is JSON-only.
